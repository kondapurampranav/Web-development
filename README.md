# WhatsApp Web UI Clone

A frontend-only clone of the WhatsApp Web interface built using **HTML** and **CSS**.

## 🔗 Live Demo
👉 https://kondapurampranav.github.io/Web-development/project-whatsapp-ui/

## 🛠 Tech Stack
- HTML5
- CSS3 (Flexbox)

## ✨ Features
- Sidebar navigation
- Chat list layout
- Dark mode UI
- Desktop-first design

## ⚠️ Disclaimer
This project is created **only for learning and practice purposes**.  
It does **not** include backend, authentication, or real messaging.
